# まとめ
| 魚種 | 現存量（2019年度比） | 特徴 |
|:---|:---|:---|
|スケトウダラ0歳 |北↓ / 南↓ |1995年以降最も低い水準|
|スケトウダラ1歳以上 |北↑ / 南↑ |特に北部で増加|
|マダラ0歳 |北↓ / 南↓ |1995年以降最も低い水準|
|マダラ1歳 |北↓ / 南↓ |特に南部で減少|
|マダラ2歳以上 |北↑ / 南↑ |震災前と同程度の水準 |
|イトヒキダラ |北↓ / 南↑ |総現存量としては減少 |
|キチジ |北→ / 南→ |個体の成長で大型化|
|ズワイガニ雌 |北↓ / 南↓ |特に北部で減少 |
|ズワイガニ雄 |北↓ / 南↓ |特に北部で減少 |
|アカガレイ|北→ / 南↓ |低水準が続いている|
|サメガレイ|北→ / 南↓ |南部では30cm前後の中型個体|
|ババガレイ |北→ / 南↓ |南北で体サイズが類似する傾向 |
